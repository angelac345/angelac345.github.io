During my internship at Gemini, I joined the Application Security team to develop an internal tool used for vulnerable code detection across Git Commits. This system utilizes Semgrep to detect texts that conform to a certain type of pattern, then generates a markdown report that organizes these information. This internship was largely remote, so seeing how people in the same team could work across different time zones was really interesting and I learned a great deal about industrial code development during the process.

PoRTaL Lab - Undergraduate Researcher

Video

Gif

Ithaca, New York; Aug 2023 - Now

I joined PoRTaL during Fall of 2023, and have been helping on developing solutions to robotics systems since then. I began by working with the Stretch RE2 robot to develop a lower-level skill set for the robot, which consists of the robot moving to the pantry, detecting a desired object, then picking and moving it to the table, as requested by the human user. I have been working on the pick policy for the robot, which is a deep learning model trained using Proximal Policy Optimization (PPO) to pick up objects. I have also been working on the reward engineering for the robot, which is the process of designing the reward function for the robot to learn the desired behavior.

I have also been working on the ablations and design of the model, which is the process of experimenting with different configurations of the model and evaluating the performance of the model with different metrics. This process is used as the basis for the design of the model, which is the process of designing and augmenting the deep learning model based on the ablations to increase the skill set of the robot.
